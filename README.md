# Karol Ignatowicz — portfolio redesign

A static GitHub Pages website inspired by a minimalist editorial finance aesthetic. It is an original design and does not copy another site's code or exact layout.

## Before publishing

1. In `index.html`, replace `https://www.linkedin.com/in/your-profile` with your real LinkedIn URL.
2. Confirm that `kontakt@karolignatowicz.pl` exists, or replace it with your preferred email.
3. Review the Auto Partner model filename and project text.
4. Keep the `CNAME` file — it connects GitHub Pages to `karolignatowicz.pl`.

## Publish with GitHub web interface

Upload all files and the entire `assets` folder to the root of `IgnatowiczKarol.github.io`, replacing the current website files. Do not delete `CNAME`.

## Local preview

Open `index.html` in a browser, or run a local HTTP server:

```bash
python -m http.server 8000
```
